Animus
======
A text to image Anima GUI for machines without CUDA, AVX2, and so on.
The old SD 1.5 version still exists as ```imagine```.
```
sudo apt-get install -y bash gir1.2-gtk-3.0 libcairo2-dev libgirepository1.0-dev libgtk-3-0 pkg-config python3-dev python3-gi python3-gi-cairo python3-pip python3-venv ; make install ; animus
```

Installs to $HOME/.local by default or set the PREFIX environment variable.
Run as ```animus```.
